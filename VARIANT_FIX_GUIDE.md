## 🔍 كيف يعمل السكريبت

### 1. إصلاح `Pattern.has_sizes`

```python
# لكل نمط:
variants_with_size = ProductVariant.objects.filter(
    pattern=pattern,
    size__isnull=False
).exists()

has_pattern_sizes = PatternSize.objects.filter(pattern=pattern).exists()

should_have_sizes = variants_with_size or has_pattern_sizes

if pattern.has_sizes != should_have_sizes:
    pattern.has_sizes = should_have_sizes
    pattern.save()
```

### 2. حساب التركيبات الممكنة

```python
# للمنتجات المعتمدة على الأنماط:
for pattern in patterns:
    if pattern.has_sizes:
        # Pattern × Color × Size
        for color in colors:
            for size in pattern_sizes:
                combinations.append((pattern, color, size))
    else:
        # Pattern × Color (بدون مقاسات)
        for color in colors:
            combinations.append((pattern, color, None))
```

### 3. إنشاء المتغيرات

```python
# التحقق من عدم وجود المتغير
if variant_not_exists:
    # حساب السعر ديناميكياً
    price = product.get_price(
        pattern_id=pattern.id,
        size_id=size.id,
        color_id=color.id
    )
    
    # الحصول على المخزون من PatternSize
    stock = pattern_size.stock
    
    # إنشاء المتغير
    variant = ProductVariant(
        product=product,
        pattern=pattern,
        color=color,
        size=size,
        price=price,
        stock=stock
    )
    
    variants_to_create.append(variant)

# الحفظ الجماعي
ProductVariant.objects.bulk_create(
    variants_to_create,
    ignore_conflicts=True
)
```

## ⚠️ ملاحظات مهمة

### ✅ آمن تماماً:

- ✅ لا يحذف أي بيانات موجودة
- ✅ يستخدم `ignore_conflicts=True` لتجنب التكرار
- ✅ يستخدم `transaction.atomic()` للتراجع عند الخطأ
- ✅ يمكن تشغيله أكثر من مرة بأمان

### 📝 المتطلبات:

1. **كل منتج يجب أن يحتوي على:**
   - نمط واحد على الأقل (`Pattern`)
   - لون واحد على الأقل (`ProductColor`)

2. **كل نمط له مقاسات يجب أن يحتوي على:**
   - مقاس واحد على الأقل في `PatternSize`

3. **المخزون الافتراضي:**
   - يتم نسخه من `PatternSize.stock`
   - إذا لم يوجد، يكون `0` (يجب تحديثه يدوياً)

def get_variant_stock(self, pattern, size):
    if pattern and size:
        try:
            ps = PatternSize.objects.get(pattern=pattern, size=size)
            return ps.stock

## 🐛 استكشاف الأخطاء

### المشكلة: "لا توجد تركيبات ممكنة"

**السبب:**
- المنتج ليس له أنماط أو ألوان

**الحل:**
1. أضف أنماط للمنتج في Admin Panel
2. أضف ألوان للمنتج في `ProductColor`

---

### المشكلة: "النمط مُعلّم كـ has_sizes=True لكن لا توجد مقاسات"

**السبب:**
- النمط مُعلّم أن له مقاسات لكن لا توجد مقاسات في `PatternSize`

**الحل:**
1. أضف مقاسات في `PatternSize` للنمط
2. أو غيّر `has_sizes` إلى `False`

---

### المشكلة: "متغيرات بدون نمط"

**السبب:**
- توجد متغيرات قديمة بدون نمط

**الحل:**
- حدد نمط لهذه المتغيرات يدوياً في Admin Panel
- أو احذفها إذا كانت غير صحيحة
